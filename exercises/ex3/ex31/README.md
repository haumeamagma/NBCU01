# Exercise 3.1 - Add a new employee

In this exercise, you will be lead the process of adding a new employee in the SuccessFactors demo instance.

Please do note that while the steps shown are comprehensive, due to time constraints on the session this process will be run by the instructors of the session for you, but this will match exactly with what is shown below.

## Exercise steps

Run through the following steps:

1. Logon to the SAP SuccessFactors with the SFSF Admin User ID
      
      <br><img src="/exercises/ex3/ex31/images/1. SFSF_Logon.png" width=70% height=70%>

2. Enter "Add New Employee" in the search field. This would bring us to the Add New Employee Wizard.
     <br><img src="/exercises/ex3/ex31/images/2. SFSF_Admin.jpg" width=70% height=70%>

3. In the Identity section, details will vary by company, but we will enter the critical employee information for our employee being added to be able to attend DCOM 2023:

- <ol type="i">
  <li>Hire Date: February 14th, 2023</li>
  <li>Company: From the drop down list - BestRun (10000)</li>
  <li>Event Reason: From the drop down list - New Hire (HIRNEW)</li>
  <li>First Name: Jim</li>
  <li>Last Name: Halpert</li>
  <li>Date of Birth: Oct 01, 1978</li>
  <li>Country of Birth: United States</li>
  <li>Person Id: DCOM2023</li>
  <li>User Name: DCOM2023</li>
 </ol>
 
 Then we click <b>Continue</b> to continue the wizard.
    <br><img src="/exercises/ex3/ex31/images/3a. New Hire.PNG" width=70% height=70%>

4. In Personal Information section, enter details as given below:

- <ol type="i">
  <li>Gender: Male</li>
  <li>Preferred Language: From drop down list - English (US)</li>
  <li>Nationality: From drop down list - United States</li>
  <li>Enter Email Information:</li>
- Email Type: Business
- Email Address: dcom_participant@sap.com
- Is Primary: From drop down list - Yes
  </ol>
  Then select continue to continue the wizard.
  <br><img src="/exercises/ex3/ex31/images/3b. New Hire.PNG" width=70% height=70%>

5. In Job Information section, enter details as given below:

Select Position: From drop down list - Inspector
In Organizational Information, No data as Cost Center.
<br><img src="/exercises/ex3/ex31/images/3d. New Hire.PNG" width=70% height=70%>

6. Scrolling down to the end of the page to Employment Details, because it would be cruel to make an employees first day be on Valentine's Day set the First Date Worked to February 15th, so they can attend this sesssion.
Click <b>Continue</b> to move the wizard forward, and then click <b>Submit</b> to finalize the Addition of the new Employee
<br><img src="/exercises/ex3/ex31/images/3e. New Hire.PNG" width=70% height=70%>

7. When the employee has been added successfully the following screen appears.
<br><img src="/exercises/ex3/ex31/images/3f. New Hire.PNG" width=70% height=70%>

You now have seen how the employee is added to the SuccessFactors system, which will trigger the event push to Event Mesh and SAP Integration Suite.
